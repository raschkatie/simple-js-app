# Pokedex App

The Pokédex app is a list of Pokémon that includes the name, height, and type of each item. 

## Features

This app utilizes a combination of:

- HTML
- CSS
- JavaScript
- Bootstrap

## Resources

The list of Pokémon was fetched from [PokéAPI](https://pokeapi.co/).
